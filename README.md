# QRCode

A Flutter plugin to generate QRCode. 
It currently only supports iOS

## 🎖 Installing

```yaml
dependencies:
  qrcode: "^0.1.0"
```

### ⚡️ Import

```dart
import 'package:qrcode/qrcode.dart';
```

## 🎮 How To Use

```dart
await Qrcode.generateQRCode('Test message'); //pass the message you want in the QRCode
```
